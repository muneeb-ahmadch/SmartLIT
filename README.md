
# Luxury Smart Home Homepage

This is the SmartLIT marketing site codebase. The original design source is available at https://www.figma.com/design/PoAt0wqsoE7w9EIvh1MWD9/Luxury-Smart-Home-Homepage.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

## Versioning

The project uses semantic versioning through the `version` field in `package.json`.

- `npm run version:patch` increments `0.1.0` -> `0.1.1`
- `npm run version:minor` increments `0.1.0` -> `0.2.0`
- `npm run version:major` increments `0.1.0` -> `1.0.0`

Each command updates `package.json` and creates a git tag when run from a clean git worktree.
